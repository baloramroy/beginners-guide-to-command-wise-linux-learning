## ⏱️ `uptime` — Show System Uptime

### Show how long the system has been running

```bash
uptime
```

> **Output:** 10:00:06 up 23:18,  3 users,  load average: 0.00, 0.00, 0.00

### Show uptime in a pretty format

```bash
uptime -p
```

> **Output:** up 23 hours, 19 minutes

### Show system boot time (yyyy-mm-dd HH:MM:SS)

```bash
uptime -s
```

> **Output:** 2025-11-16 10:41:54
