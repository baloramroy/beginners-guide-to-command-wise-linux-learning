## Hostname Commands

### Print the system hostname

```bash
hostname
```

> **Output:** centosdomain

### Print long host name (FQDN)

```bash
hostname -f
```

> **Output:** centosdomain.local

### Print host address information

```bash
hostname -i
```

> **Output:** fe80::20c:29ff:fe35:fec2%ens34 192.168.231.213


### Print host IP information

```bash
hostname -I
```

> **Output:** 192.168.231.213
