### Show the calendar for a specific year (2025)

```bash
cal 2025
```

### Display the current month (default)

```bash
cal
```

### Display previous, current, and next month (3 months)

```bash
cal -3
```

### Display *n* (5) months starting from the current month

```bash
cal -n 5
```

### Display calender using a vertical layout

```bash
cal -v
```

### Display a full year calendar

```bash
cal -y
```

### Display week numbers before calender (US or ISO-8601)

```bash
cal -w
```
