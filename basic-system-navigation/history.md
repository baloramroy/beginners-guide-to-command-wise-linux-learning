# history - Shell Command History

Display or manipulate the history list

## Example:
  - **List of history command**
  ```bash
  history
  ```
  **Output:** View all the previous command history

- **Enable shell history**
  ```bash
  set -o history
  ```
  **Output:** Enable history option for the shell

- **Disable shell history**
  ```bash
  set +o history
  ```
  **Output:** Disable history option for the shell
